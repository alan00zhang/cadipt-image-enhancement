# cadipt-image-enhancement
Image enhancement algorithm for upscaling and increasing visual fidelity using second derivative enhancement with deconvolution for 8x upscaling.
# Installation
Download the .exe file and place it anywhere you like.
Place images (can be .png, .jpg, .dat, .mat) into a folder named "input" in the same location as the executable.
Run the .exe and you can find your results in the /output folder.
